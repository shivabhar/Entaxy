# Entaxy
Entaxy
Entaxy is the application which is automated using Selenium, this web app is meant to help users to better understand their finances and how to manage those.

Page Object Model(POM) is a design pattern to create Object Repository for web UI elements. 
This model has different packages for Page Objects, Utilities, Test Data, Test script 
Testdata: This holds the data for input purpose 
PageObjectModel: This stores the web elementsand page Objects 
TestScript: This stores the test commands to run multiple test cases (test suite)
Util: This hold the required drivers to automate the web app in different browsers(here it is google chrome!)
